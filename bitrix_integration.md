## Файлы:
1. `property_cards.html` - HTML версия карточек
2. `bitrix_properties.json` - данные для импорта в Bitrix24
3. `debug_properties.json` - полные данные с фото

## Следующие шаги для Bitrix24:

1. **Импорт данных в CRM:**
   - Используйте `bitrix_properties.json` 
   - Настройте соответствие полей

2. **Настройка карточек в Bitrix24:**
   - В настройках CRM -> Настройки полей
   - Добавьте поле для фотографий
   - Настройте шаблон отображения

3. **Автоматизация:**
   - Настройте регулярный парсинг
   - Авто-обновление данных
   - Уведомления о новых объектах

## Поля для Bitrix24:
- NAME - Название объекта
- ADDRESS - Адрес  
- PRICE - Цена в USD
- PRICE_M2 - Цена за м²
- ROOMS - Количество комнат
- AREA - Площадь
- DISTRICT - Район
- PHOTOS - Массив фото URL
- PHOTO_COUNT - Количество фото
